2 Заголовок-менший від 1.
3 Заголовок-середній.
4 Заголовок-менший від 3.
5 Заголовок-менший від 4.
6 Заголовок-найменший.


Тег align-розміщення по центру зліва або зправа.

Атрибут align використовується для того щоб розміщати елементи на сторінці.

Форматування тексту

Усього є 6 тегів для форматування.
1-ший тег /b/- жирний шрифт для тексту.
2-ий тег /i/- косий шрифт для тексту.
3-ій тег /u/- підкреслений шрифт для тексту.
4-тий тег /s/-викреслений шрифт для тексту.
5-тий тег /sup/-індекс підкреслений зверху.
6-тий тег /sub/-індекс підкреслений знизу.
